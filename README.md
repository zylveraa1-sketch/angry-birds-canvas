# Angry Birds — 2D Canvas

Рогатка, физика блоков, свиньи, 5 уровней, счёт и экран победы/поражения.

## Запуск

**Windows:** `start-game.bat`

```bash
npm install
npm run dev
```

http://localhost:5173/

## Сборка

```bash
npm run build
npm test
```

## Управление

- Потянуть птицу на рогатке и отпустить
- Двойной клик после запуска — способность жёлтой или чёрной птицы
